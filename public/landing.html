vocalize

Create and edit websites just by talking to them. This project is built for the Facebook Developer Circles Community Challenge.

try it

What is vocalize?

Create

Build, edit, and deploy websites live just by talking to them.

Code

Create web frontends in HTML & CSS through natural language.

Speed

Quickly prototype a frontend or integrate the code into larger projects.

Why do I need vocalize?

Do you build websites? Do you work at a marketing agency that builds websites for clients? Or, do you work for a company that occasionally makes its own websites? Vocalize can significantly speed up and de-frustrate this process for programmers and more normal humans alike.

How does it work?

Talk to it. Tell vocalize you want to add three sections to the site. Tell it you want a main banner section with a background image of space. Tell it you'd like to add the Gettysburg Address text alongside pictures of unicorns.

When you are done, deploy the website effortlessly. Alternatively, export the code (simple HTML, CSS, Javascript) to incorporate your creation into another project.
